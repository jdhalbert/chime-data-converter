Chime & Marcus Transaction Data JSON to CSV Converter

To get Chime JSON data, go to chime.com, press F12, and log in. In the Network->Fetch/XHR tab, find the "graphql" response that contains JSON-formatted transaction data.

To get Marcus JSON data, go to marcus.com, press F12, and load the page with transaction detail. In the Network->Fetch/XHR tab, find the response called "cos/?operations=Transations".

Copy and paste the entire transaction data from either service into the box below.

Processing is done client-side. No data leaves your machine.
